# USEsteampics

Локальный микро-сервер с графическим интерфейсом для работы в связке с пользовательским скриптом **[Ultimate Steam Enhancer (U.S.E.)](https://github.com/0wn3dg0d/Ultimate-Steam-Enhancer)**.

## Зачем это нужно?
Обычные браузерные скрипты (Tampermonkey) не могут напрямую подключаться к скрытой базе данных Steam Network (PICS) из-за строгой политики безопасности браузеров (CORS) и специфического шифрования протоколов Steam. 

**Для чего это на практике?**
Если вы хотите купить подарить игру другу из другого региона, вы можете столкнуться со **строгими региональными ограничениями**, когда игру нельзя даже подарить в определённый регион. Магазин Steam не показывает эту информацию на странице игры. Раньше, чтобы убедиться в отсутствии строгих блокировок, приходилось вручную проверять ID пакетов каждой игры на сторонних ресурсах вроде SteamDB.

**USEsteampics** автоматизирует этот процесс. Это легковесное портативное приложение анонимно подключается к Steam, вытягивает закрытые манифесты пакетов и отдаёт их скрипту U.S.E. 

Вам достаточно один раз указать нужный регион (например, регион вашего друга) в настройках скрипта. Если издатель наложил жёсткие ограничения, скрипт покажет предупреждающую плашку прямо рядом кнопкой «В корзину», избавив вас от необходимости искать информацию вручную.

## Как использовать
1. Перейдите в раздел [Releases](../../releases/latest) (справа на странице).
2. Скачайте последнюю версию `USEsteampics.exe`.
3. Запустите файл (программа откроется и её можно будет свернуть в системный трей).
4. Зайдите в настройки скрипта **Ultimate Steam Enhancer** на сайте Steam, включите опцию `Проверка региональных ограничений` и выберите регион для проверки.
5. Готово! Скрипт U.S.E. автоматически найдет сервер в фоне и начнет показывать скрытые региональные блокировки пакетов.

> 🔒 **Безопасность:** Программа работает исключительно локально на вашем ПК (порт 8080), ничего не передает третьим лицам и логинится в Steam как анонимный пользователь, не требуя ввода вашего логина и пароля. Исходный код полностью открыт.
